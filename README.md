# GGO

Georgia Golden Olympics

https://georgiagoldenolympics.org/

11ty, Pug, Gulp, Less

## Build
```
npm run build
```
builds to `_site`

## Dev
```
npm run dev
```
spins up http://localhost:8081 and watches for changes

## Clean
```
npm run clean
```
removes 11ty and less build folders, automatically run before `dev` and `build`
